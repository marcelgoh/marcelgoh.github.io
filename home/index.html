  • Suppose $G$ acts on a set $S$ and let $s\in S$. The orbit $O_s$ of $s$ is the set of all elements $s'\in S$ such that $s' = g(s)$ for some $g\in G$. The stabiliser $G_s$ of $s$ is the set of all $g\in G$ such that $g(s) = s$. If $O_s = S$ for some $s\in S$, then we say $G$ acts transitively on $S$.
